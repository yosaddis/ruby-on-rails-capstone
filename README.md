## ğŸ’» Getting Started <a name="getting-started"></a>

To run the project locally, follow these steps.

### Prerequisites

In order to run this project you need to:

- Install [Git](https://git-scm.com/)
- Install [Ruby](https://www.ruby-lang.org/en/)
- Install Rails gem `gem install rails`
- Install a code editor, We suggest [VsCode](https://code.visualstudio.com/)

### Setup

Clone the repo into your local machine:

```bash
git clone https://github.com/yosaddis/ruby-on-rails-capstone
```

Install the dependencies using bundler:

```bash
bundle install
```

Do this to for database migrations :

```bash
rails db:drop
```

```bash
rails db:create
```

```bash
rails db:migrate
```

### Usage

To start you can open your terminal and in the folder write

```bash
rails server
```
